# Duck Race Randomizer

Vite + React duck race app with a real WebGL/Three.js race track. Ducks are rendered as 3D mesh models with body, head, beak, wings, legs, accessories, labels, shadows, water lanes, and finish effects.

## Deploy
Upload the whole folder contents to GitHub, then import the repo into Vercel.

## Local checks

```sh
npm install
npm run build
npm audit
```
